# Pasta public/images

Esta pasta armazena imagens públicas usadas pelo projeto.

- Local: `public/images`
- Uso: coloque imagens que podem ser servidas publicamente (ex.: assets, logos, imagens de exemplo).
- Convenções de nomes: use minúsculas, hifens ou underscores, sem espaços. Ex.: `my-image.png`.
- Formatos recomendados: `.png`, `.jpg`, `.webp`, `.svg`.
- Tamanho: otimize imagens antes de commitar.
- Atenção: não coloque imagens sensíveis ou privadas aqui.
- Como referenciar: `/public/images/nome-do-arquivo.ext` ou caminho relativo conforme seu servidor.

Se quiser, posso:
- Adicionar um arquivo `.gitignore` para ignorar imagens temporárias.
- Criar uma GitHub Action para otimizar imagens ao fazer push.
